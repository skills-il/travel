# Travel Skills

AI agent skills for Israeli travelers: trip planning in Israel and abroad, flights from Ben Gurion, visas, passport logistics, and travel advisories.

Part of [Skills IL](https://github.com/skills-il), curated AI agent skills for Israeli users.

## Skills

Skills are listed and installable from the [Agent Skills directory](https://agentskills.co.il/he/categories/travel).

## Install

```bash
# Claude Code, install a specific skill
claude install github:skills-il/travel/<skill-slug>

# Or clone the full repo
git clone https://github.com/skills-il/travel.git
```
